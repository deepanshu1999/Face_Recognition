# Face_Recognition
This is a project on Image recognition. It Identifies basic features of human that include
-Full Face
-Eyes
-Upper Body
-Lower Body
-Smile
# Tech
- [OpenCV](https://github.com/opencv/opencv)
- [Haarcascade](https://github.com/opencv/opencv/tree/master/data/haarcascades)
- [Python]
- Jupyter Notebook
